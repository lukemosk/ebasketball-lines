# EBASKETBALL TRACKER - QUICK COMMAND REFERENCE

## Main Operations

### Start Tracking
python integrated_quarter_tracker.py --monitor --adaptive    # Track openers, quarters,  results
python live_db_monitor.py                 # Database monitor (separate terminal)

### Core Analysis
python middling_analysis.py               # Opening line middle analysis
python quarter_middling_analysis.py       # Quarter-based middle analysis
python quarter_middling_analysis.py --quarters 2     # Q2 only (halftime)
python quarter_middling_analysis.py --quarters 1,3   # Q1 and Q3 only

### Detailed Analysis
python middle_progression_analysis.py     # Clean hit rate comparison
python game_progression_analysis.py       # Game-by-game line movement

### Visualization Tools
python dashboard.py                       # Interactive dashboard (http://127.0.0.1:8050)
streamlit run streamlit_monitor.py        # Live monitor (http://localhost:8501)
python visual_analysis.py                 # Static charts (heatmap, boxplots)

## Database Management

### Check Status
python check_event_status.py              # See game statuses
python check_captures.py                  # View recent captures

### Fix Issues
python fix_missing_results.py             # Add missing result records
python update_old_live_games.py           # Update stuck "live" games
python cleanup_bad_timing.py              # Remove bad quarter captures

### Backup
python backup_database.py                 # Create timestamped backup

## Utility Scripts

### Testing
python -m src.betsapi                     # Test API connection
python test_live_betting_lines.py         # Test line extraction

### Environment
.venv\Scripts\activate                    # Activate virtual environment
pip install -r requirements.txt           # Install dependencies

## Key Files

- .env                                    # API key and configuration
- data/ebasketball.db                    # Main database
- integrated_quarter_tracker.py          # Core tracking logic
- src/betsapi.py                         # API client

## Operating Hours

Best times: 9 AM - 11 PM EST (most games active)
Game duration: ~20 minutes (4x5 minute quarters)

## Troubleshooting

1. No games showing: Check if within operating hours
2. Storage issues: Delete .db-wal and .db-shm files
3. Missing imports: .venv\Scripts\activate then pip install
4. API errors: Check .env has valid BETSAPI_KEY
